# BUG TRACK TEST APP



front: react, clojurescript, rum, milligram  
back: scala, playframework, slick  

## How to run

```
slick.dbs.default.db.user="postgres"
slick.dbs.default.db.password="iddqd"
```

Start the Play app:

```bash
sbt run
```

And open [http://localhost:9000/](http://localhost:9000/)
